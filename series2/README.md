# Sentiment Analysis API

## Project Description
This is part of the SE Sz1 Sem2 25/26 course project, implementing a sentiment analysis API using Hugging Face Transformers.

## Features
- Text sentiment analysis (Positive/Negative)
- RESTful API interface
- Automatic model download and loading
- Comprehensive test coverage
- CI/CD pipeline
